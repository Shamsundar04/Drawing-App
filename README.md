# Drawing Application

A modern, responsive web-based drawing application built with HTML5, CSS3, and JavaScript. This app provides a simple yet powerful interface for creating digital drawings with various tools and options.

## Features

- **Multiple Drawing Tools**: Pencil, Eraser, Line, Rectangle, Circle, and Text
- **Customizable Colors**: Full color palette with custom color picker
- **Adjustable Brush Size**: Control the thickness of your drawing tools
- **Canvas Operations**: Clear canvas, save drawings, and load images
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Real-time Preview**: See tool size and color selection before drawing

## Live Demo

You can try the application directly by opening `index.html` in your web browser.

## Installation

No installation required! This is a client-side web application.

1. Download the project files
2. Open `index.html` in any modern web browser
3. Start drawing!
